# .dotfiles

Setup my Linux dev environment in one single command.

## Install

1. Clone the repository in your home directory.

```
git clone git@github.com:Edouardbozon/.dotfiles.git ~/.dotfiles
```

2. Run the script.

```
~/.dotfiles/install.sh
```
